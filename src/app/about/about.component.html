About CoalSand



CoalSand is a crud based single page web application.It is made using MEAN stack.MEAN means MongoDB ,Express.js , Angular, Node.js.

  • MongoDB is used as DataBase Managemment System.
  • Angular is used as the framework.
  • Node.js is used to run the Javascript files of Angular.
  • Express.js is a Node.js Framework used to build and run the rest api which connects the database and the webapp.
  • We have used Mongoose as Object-Relational Mapping Library for Api

CoalSand provides the admin side user interface of a shop management system. In Coalsand a user can add his product , update a previously added product , delete a product and also search a product.
